CV8 Overview

CV8 is an open, NMRA aligned control ecosystem designed to give modellers a clear, modern, and interoperable alternative to legacy command control systems. It exists to unify wiring, signalling, and accessory control under a single, transparent standard that anyone can implement, extend, or build hardware for without licensing barriers. At its core, CV8 solves the fragmentation problem: too many proprietary buses, too many incompatible accessory interfaces, and too many layouts locked into one manufacturer’s worldview.

It’s built for modellers, makers, SIG members, and manufacturers who want reliability, clarity, and long term stability. CV8 doesn’t replace MSS, CMRInet, or LCC — it sits alongside them as a practical, open, and implementation friendly layer. MSS defines movement; CMRInet defines communication; LCC defines distributed nodes. CV8 provides the missing piece: a clean, unified, open hardware and wiring standard that ties the whole ecosystem together.
